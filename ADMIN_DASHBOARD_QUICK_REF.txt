## âš™ï¸ Settings Reference

### Framework Settings
- **Debug Mode** - Enable detailed logging
- **Enable Caching** - Cache components for performance
- **Server-Side Rendering** - Render on server (faster)
- **Lazy Loading** - Load components on scroll
- **Cache Duration** - How long to cache (seconds)

### Advanced Settings
- **SPA Root Element** - Container CSS selector (e.g., #app)
- **REST API Prefix** - API endpoint (e.g., wpway/v1)
